Petlyst is a senior-year project developed at Bilkent University (CTIS) that aims to simplify and modernize the interaction between pet owners and veterinarians. The platform offers core features such as appointment scheduling, digital health record tracking, and real-time video consultations. Additionally, we integrated a lightweight AI assistant powered by a Large Language Model to support users with basic guidance and reminders.
